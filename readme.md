# Stopwatch Web Application ⏱️

A simple, interactive stopwatch web app built using HTML, CSS, and JavaScript.

# Features
- Start / Pause / Reset stopwatch
- Lap time recording
- Real-time time display (with milliseconds)
- Clean and responsive UI
